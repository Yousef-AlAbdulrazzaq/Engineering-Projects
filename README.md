--------------------------------------------------------
PROJECTS
--------------------------------------------------------
 
1. Digital Robotic Sensor on FPGA
   Folder: /digital-robotic-sensor-fpga
   Description: 8-function FPGA module in Verilog HDL driving a 7-segment display from
   dual 3-bit sensor inputs
   Tools: Verilog, Quartus II, Altera Cyclone IV E DE2-115
   Course: CE/EE 270
 
2. Vehicle Anti-Theft System with Engine Lock
   Folder: /vehicle-anti-theft-system
   Description: Vibration-triggered anti-theft system that alerts the owner and disables
   the engine
   Tools: Arduino, sensors, actuators
   Course: ENGR 132
 
3. Industrial Load Analysis & Power Factor Improvement
   Folder: /industrial-load-power-factor
   Description: Simulated a 5-branch industrial load system and designed a capacitor bank
   to raise power factor from 0.715 to 0.95
   Tools: OrCAD PSpice
   Course: EECE 202
 
4. Smart Parking Occupancy & Warning Indicator
   Folder: /smart-parking-indicator
   Description: All-analog vehicle occupancy detection with IR sensors, Schmitt trigger
   comparator, and RC timing
   Tools: Op-amps, discrete components
   Course: EECE 255
 
5. Battery Range Indicator
   Folder: /battery-range-indicator
   Description: Window-comparator circuit classifying battery charge into
   low/normal/critical via LEDs
   Tools: OrCAD, op-amps, transistors
   Course: EE-CE 201
 
6. Door Security Alarm (BJT Switching)
   Folder: /door-security-alarm
   Description: Discrete-transistor door alarm converting a magnetic sensor trigger into
   an audible/visual alert
   Tools: BJTs, breadboard
   Course: EE-CE 208
 
7. Feigenbaum Number Estimation - Chaotic Oscillator
   Folder: /chaotic-oscillator-feigenbaum
   Description: Measured period-doubling bifurcation in a diode-inductor circuit to
   estimate the Feigenbaum constant
   Tools: Function generator, oscilloscope
   Institution: University of Southampton
